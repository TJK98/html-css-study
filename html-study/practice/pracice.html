HTML의 정의

HTML의 정의 HTML (HyperText Markup Language)는
웹 페이지의 구조(Structure)를 정의하기 위한 마크업 언어입니다.
HTML, CSS, JavaScript 등 웹 기술에 대한 공식 문서와 예제를 제공하는 웹사이트는 MDN을 클릭하세요.

텍스트 요소

  • 제목 태그 (h1~h6)와 콘텐츠 계층 구조

    • 콘텐츠의 제목이나 주제를 계층적으로 표현하는 태그로, 숫자가 작을수록 중요도와 글씨 크기가 크다.

  • 단락 태그 p

    • 문장을 하나의 단락(문단)으로 묶어주는 태그로, 브라우저에서 자동으로 줄바꿈과 여백을 적용한다.

  • 줄바꿈 태그 br

    • 강제 줄바꿈을 삽입할 때 사용하는 태그로,
    문장을 끊지 않고 줄만 바꿔준다.

  • 구분선 태그 hr

    • 내용 사이에 주제의 전환이나 구분을 나타낼 때 사용하는 수평선(horizontal rule) 태그로,

    시각적으로 선을 삽입한다.

  • 엔티티 코드

    • <브라우저에서 특수 문자(예: <, >, &, 공백 등)를 올바르게 표시하기 위해 사용하는 문자 코드 표현 방식이다.>

  1. 순서 있는 목록 태그

    2. <ol>은 항목들을 1, 2, 3... 과 같은 번호로 자동 정렬해 주는 순서 있는 목록을 만드는 블록 요소입니다.

  • 목록 항목 태그

    • <li>는 <ul>이나 <ol> 내부에서 실제 항목 하나를 나타내는 블록 요소입니다.

표와 폼

    table

    표 전체를 감싸는 태그로, 행과 열 구조를 정의하는 컨테이너 역할을 한다.

    tr

    표의 한 가로줄(행, row)을 나타내는 태그로, 그 안에 셀들이 위치한다.

    td

    표의 일반 데이터 셀(열, column)을 나타내는 태그로, 실제 값을 담는다.

    th

    표의 헤더 셀로, 굵은 글씨와 가운데 정렬이 적용되는 제목용 셀이다.

    table tr td th
    폼 요소 form

    사용자의 입력 데이터를 서버로 전송하기 위한 입력 영역(Form)을 정의하는 태그이다.

    input 태그와 type 속성

    사용자의 다양한 입력을 수집하기 위해 사용하는 태그로, type 속성에 따라 동작과 모양이 달라진다.
